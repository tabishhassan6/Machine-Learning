Spam Detection using Machine Learning

# Project Overview
This project aims to classify SMS messages as Spam or Ham (Not Spam) using Machine Learning algorithms.

# Objective
To build a classification model that can automatically detect spam messages and reduce unwanted communication.

# Dataset
The dataset contains SMS messages labeled as:
- 0 → Ham (Not Spam)
- 1 → Spam

# Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn

# Models Implemented
- Logistic Regression
- Naive Bayes

# Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1-Score

# Project Workflow
1. Data Loading
2. Data Cleaning
3. Text Preprocessing
4. Feature Extraction (CountVectorizer)
5. Model Training
6. Model Evaluation

# Results
The model successfully classifies spam and non-spam messages with good accuracy and recall.
